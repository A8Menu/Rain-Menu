Rain Mod Menu

Works on GTA V single player Version 1.68

Download the menu here.

  • Install Script Hook V: https://www.dev-c.com/gtav/scripthookv/
  • Installation:
    1. Copy ScriptHookV.dll to the game's main folder, i.e. where GTA5.exe is located.
    2. In order to load ASI plugins, you need to have an ASI loader installed. You can download it separately or use the latest version that comes with this distribution (dinput8.dll). You must delete any previous ASI loader version (dsound.dll) if you have one installed.
    3. There will be a native trainer.ASI; do not drag that in.
  • Install Script Hook V .NET: https://github.com/scripthookvdotnet/scripthookvdotnet#readme
  • Download the menu.
  • Drag everything into your GTA folder. Create a 'scripts' folder in your GTA folder and put the rain.dll in it.
  • Press F5 to open the menu. The menu is constantly being updated, so make sure to always check for updates. Join the Discord for support and announcements about the menu: https://discord.gg/pR2C49hPp2
