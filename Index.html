Rain Mod Menu

Works on GTA V Online Version 1.89

Download the menu here.

  • Put the script in the scripts folder of YimMenu.
  • You can find the YimMenu folder by opening the RUN window (Windows key + R) and typing %appdata% and pressing enter.
  • Then look for the YimMenu -> scripts folder and put the script in there.
  • OR you can open YimMenu, go to Settings -> Lua Scripts and click on the Open Lua Scripts Folder button. Then put the script in there.
  • Launch GTA V and open YimMenu if you haven't already.
  • Click on the Reload All button if the script does not appear.
